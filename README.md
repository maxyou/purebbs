# purebbs
forum/bbs using "react+koa2+mongodb"

done:

- post/delete/update/settop/like/anonymous
- vote/lineup
- login/register/change pwd/email reset pwd/github login
- user role: admin/bm/user
- upload avatar
- english/chinese
- markdown
- board configure
- save user setting when logout
